# Screenshots

This folder contains app screenshots for the README and documentation.

## Expected Structure

```
docs/screenshots/
├── onboarding.png          # Onboarding / splash screen
├── login.png               # Login screen
├── dashboard.png           # Dashboard / topic list
├── topic-details.png       # Topic details with progress arcs
├── summary.png             # Summary reader screen
├── summary-audio.png       # Audio playback screen
├── challenge-quiz.png      # Quiz challenge run
├── challenge-hangman.png   # Hangman challenge run
├── challenge-matrix.png    # Matrix challenge run
├── challenge-finished.png  # Challenge score screen
├── calendar.png            # Calendar view
├── profile.png             # Profile / menu screen
└── dark-mode.png           # Dark mode showcase (optional)
```

## Guidelines

- Format: **PNG**, max 800px wide
- Capture on **iPhone 15 Pro** (Xcode Simulator) or real device
- Use light mode for primary screenshots; add dark-mode variant if available
- No personal/sensitive data visible in screenshots
- Filename must match the key used in the README `![alt](docs/screenshots/name.png)` links
