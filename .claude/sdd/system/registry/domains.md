# Domain Registry

## Active Domains

### auth

- login
- signup
- password reset
- session lifecycle
- biometric access

### dashboard

- home summaries
- widgets
- KPI cards
- feed surfaces
- personalized entry states

## Candidate Domains

- profile
- payments
- notifications
- subscription
- onboarding

## Naming Rules

- domain names are lowercase kebab-case
- epic names are lowercase kebab-case
- work-item names use external id when available, for example `us-123-login`
