# SDD Creation

Use this skill for feature design, architecture proposals, migrations, and execution blueprints.

## Expected output

- problem framing
- scope and exclusions
- architecture and decisions
- phased rollout
- risks and rollback
- verification and acceptance

Use templates from `assets/` when creating new SDDs.
